# Serverless LLM

LLM playground running on GitHub Actions, exposed via Cloudflare Tunnels. Static frontend on GitHub Pages.

## Commands

- `make build` — build frontend
- `make lint` / `make format` — Python code quality
- `make tunnels-secret` — collect tunnel tokens and set TUNNELS_JSON secret
- `make inference MODEL=<name>` — launch one inference server
- `make up` / `make down` — launch or cancel all inference servers

## Architecture

`app/chat/frontend/` — Vite + React static frontend (deployed to GitHub Pages)
`app/*/` — inference server containers (run on GitHub Actions)
`config/models.py` — single source of truth: ports, tunnel IDs, model metadata
`docker-compose.yml` — local development
`tunnels.json` — flat `{model: token}` map (gitignored, generated by `make tunnels-secret`)
