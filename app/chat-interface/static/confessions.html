Confessions: Train for honesty, not just polish

The Problem: Models are trained to be helpful and polished, but this can mask hidden rule breaks, hallucinations, or reward-hacking attempts.

The Solution: Inspired by OpenAI's research, this mode requests a second response where the model self-reports how it followed every instruction. The main answer is scored on usefulness, while the confession is rewarded only for honesty.
