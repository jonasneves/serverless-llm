Confessions Mode: Train for honesty, not just polish

Inspired by OpenAI's "Training LLMs for Honesty via Confessions", this mode requests a second response where the model self-reports how it followed every instruction. The main answer is scored on usefulness, while the confession is rewarded only for honesty—surfacing hidden rule breaks, hallucinations, or reward-hacking attempts.

Read the research paper · Proof-of-concept model hits only 4.4% false negatives across adversarial tests.

  • Hallucination false negatives: 7.6%
  • Instruction-following false negatives: 2.5%
  • Scheming false negatives: 2.9%
  • Hacking confession rate: 89.7%
